Detailed Description

Helper class used by CRadialBasisFunction to calculate the interpolation weights. This does not inherit from CSysMatrix because: it is a dense format rather than block sparse; as the interpolation is done on a single core there are no methods for communication. The code can be compiled with LAPACK to use optimized matrix inversion and multiplication routines. CPPFLAGS="-DHAVE_LAPACK" LDFLAGS=-L/path/to/lapack_lib LIBS="-llapack -lrefblas -lgfortran".
