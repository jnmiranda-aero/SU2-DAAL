Parameters
[in]val_VTK_Type- Type of the element using the VTK convention.
[in]val_nPoly- Polynomial degree of the element.
[in]val_constJac- Whether or not the Jacobians are constant.
[in]config- Object, which contains the input parameters.
[in]val_orderExact- Default argument. If specified, it contains the order of the polynomials that must be integrated exactly by the integration rule.
[in]rLocSolDOFs- Default argument. If specified, it contains the parametric r location of the solution DOFs.
[in]sLocSolDOFs- Default argument. If specified, it contains the parametric s location of the solution DOFs.
[in]tLocSolDOFs- Default argument. If specified, it contains the parametric t location of the solution DOFs.
