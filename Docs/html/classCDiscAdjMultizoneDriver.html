◆ Has_Deformation

su2vector<bool> CDiscAdjMultizoneDriver::Has_Deformation
protected

True if iZone has mesh deformation (used for lazy evaluation of TRANSFER tape section).

Individual cross-terms of the coupled problem, 5D array [iZone][jZone][iSol](iPoint,iVar). The column sum, i.e. along all iZones for each jZone, gives the External (total cross-term) for jZone, we need to store all terms to have BGS-type updates with relaxation.
