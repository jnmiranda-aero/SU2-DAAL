Detailed Description

template<typename Index_t>
class CCompressedSparsePattern< Index_t >

A simple class to store adjacency information in a compressed format suitable for sparse matrix operations. If built for row-major storage the inner indices are column indices and the pattern should be used as (row,icol), otherwise as (col,irow).
