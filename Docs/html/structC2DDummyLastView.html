Detailed Description

template<class T>
class C2DDummyLastView< T >

Helper class, adds dummy trailing dimension to a reference of a vector object making it a dummy matrix.

Note
The constness of the object is derived from the template type, but we allways keep a reference, never a copy of the associated vector.
