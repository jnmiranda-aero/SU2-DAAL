56 /*!
57  * \class CSysSolve
58  * \ingroup SpLinSys
59  * \brief Class for solving linear systems using classical and Krylov-subspace iterative methods
60  *
61  * The individual solvers could be stand-alone subroutines, but by
62  * creating CSysSolve objects we can more easily assign different
63  * matrix-vector products and preconditioners to different problems
64  * that may arise in a hierarchical solver (i.e. multigrid).
65  *
66  * The methods of this class are designed to be called by multiple OpenMP threads.
67  * Beware of writes to class member variables, for example "Residual" should only
68  * be modified by one thread.
