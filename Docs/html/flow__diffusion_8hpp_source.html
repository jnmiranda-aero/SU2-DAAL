32 /*!
33  * \class CAvgGrad_Base
34  * \brief A base class for computing viscous terms using an average of gradients.
35  * \details This is the base class for the numerics classes that compute the
36  * viscous fluxes for the flow solvers (i.e. compressible or incompressible
37  * Navier Stokes). The actual numerics classes derive from this class.
38  * This class is used to share functions and variables that are common to all
39  * of the flow viscous numerics. For example, the turbulent stress tensor
40  * is computed identically for all three derived classes.
41  * \ingroup ViscDiscr
42  * \author C. Pederson, A. Bueno, F. Palacios, T. Economon
43  */
