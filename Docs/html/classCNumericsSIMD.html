Interface for edge flux computation.

Parameters
[in]iEdge- The edges for flux computation.
[in]config- Problem definitions.
[in]geometry- Problem geometry.
[in]solution- Solution variables.
[in]updateType- Type of update done on vector and matrix.
[in]updateMask- SIMD array of 1's and 0's, the latter prevent the update.
[in,out]vector- Target for the fluxes.
[in,out]matrix- Target for the flux Jacobians.
Note
The update mask is used to handle "remainder" edges (nEdge mod simdSize).
