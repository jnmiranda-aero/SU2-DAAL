Detailed Description

template<class Scalar_t, size_t N = preferredLen<Scalar_t>()>
class simd::Array< Scalar_t, N >

A simple SIMD type relying on implicit vectorization, i.e. done by the compiler, explicitly vectorized specializations are defined after.

Note
This class gets its math operator overloads from CVecExpr, the specializations do not use expression templates, IF YOU NEED A NEW FUNCTION, define it both in vector_expressions.hpp and in special_vectorization.hpp.
