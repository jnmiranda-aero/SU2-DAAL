Detailed Description

template<typename Index_t, class Scalar_t, StorageType Store, size_t AlignSize, size_t StaticRows, size_t StaticCols>
class container_details::AccessorImpl< Index_t, Scalar_t, Store, AlignSize, StaticRows, StaticCols >

Base accessor class and version of template for both sizes known at compile time.

The actual container inherits from this class, this is to reduce the number of methods that need to be redefined with each size specialization.
