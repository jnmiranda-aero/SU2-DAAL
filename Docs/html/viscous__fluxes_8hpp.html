class  CNoViscousFlux< NDIM >
 Numerics classes that accept a compile-time decorator should use this class template as a "do-nothing" decorator and as a link to the interface when they are not being decorated. Compile-time decoration works by specifying the base class as a template parameter. Then the class being decorated should call a method of its base class to add some contribution to the flux/source and Jacobians just before writting the results to CSysVector and CSysMatrix. The mechanism can be used to chain any number of classes at compile-time, but its main purpose is to combine convective and viscous fluxes in the most (nearly) efficient way. More...
