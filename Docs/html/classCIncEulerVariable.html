Detailed Description

Class for defining the variables of the incompressible Euler solver.

Note
Primitive variables (P, vx, vy, vz, T, rho, beta, lamMu, EddyMu, Kt_eff, Cp, Cv)
Gradients of primitives (P, vx, vy, vz, T, rho, beta)
Author
F. Palacios, T. Economon, T. Albring
